
## Install
Clone the project

![Screenshot of the App.](/public/demo.png)

```bash
git clone git@github.com:BrunoPoiano/screen-recorder.git
```
```bash
cd screen-recorder
```
```bash
npm i
```
```bash
npm run build
```
**To execute the code**
```bash
npm run electron-start
```
**To build the executable**
```bash
npm run electron-forge-make
```
